Bonsai Assistant Project

An automated irrigation and monitoring system for bonsai trees

Project Overview

The Bonsai Assistant is an IoT project I developed to automate the care of my bonsai trees. Using a Raspberry Pi, moisture sensors, and a custom Python application, the system monitors soil moisture levels and automatically waters the plants when needed.

Key Features

  • Real-time soil moisture monitoring
  • Automated watering based on moisture thresholds
  • Data logging and visualization
  • Mobile notifications for watering events
  • Web interface for remote monitoring

Technical Stack

Raspberry Pi Python GPIO Moisture Sensors Relay Module Flask SQLite

Challenges & Solutions

One of the main challenges was calibrating the moisture sensors for different soil types. I solved this by implementing a calibration mode that allows users to set custom thresholds for each plant. Additionally, preventing over-watering required implementing a cooldown period between watering cycles.

Results

The system has been running successfully for over 6 months, maintaining optimal soil moisture levels for my bonsai collection. It has reduced daily maintenance time and improved plant health through consistent watering schedules.
